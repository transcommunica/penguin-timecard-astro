<p><a href="https://app.tc-timecard.com/">こちら</a>のリンクか、右上の「ダッシュボード」ボタン（スマホの場合はメニュー内の「ログイン」ボタン）からログインしてください。</p>
<p>手元にiPadは必要ありません。管理者は好きなときにデータを閲覧できます。<br />
「オフィスにiPadを置いてきた」<br />
「いますぐ見たいのに」<br />
「iPadが小さくて見づらい」<br />
もうそんな心配はありません！</p>
<h3>特徴</h3>
<ul>
<li>WEBから好きなときに閲覧可能</li>
<li>大きな画面で見れる！</li>
<li>PDF、エクセルもダウンロードできる</li>
</ul>
<h4>機能詳細</h4>
<ul>
<li>ログインしたら、勤怠管理のサマリーが見れます</li>
<li>社員名をクリックすると詳細が見れます</li>
<li>過去の記録も閲覧できます</li>
<li>PDF、エクセルもダウンロードできます</li>
</ul>
<h4>機能制限</h4>
<p>みなさまのご要望にお応えすべく、開発者たちが日々頑張っておりますが、現時点では、以下のような制限があります。</p>
<ul>
<li>アプリのデータ反映は1日１回です。最後に反映された時刻は画面中央上で確認できます。<br />
いますぐ反映させたい場合は、アプリ管理画面の「バックアップ」から「今すぐバックアップ」をクリックしてください。今後改善予定です。</li>
<li>残業時間などのアプリ内での計算とクラウド版での計算はそれぞれ行っております。万が一、数字が一致しないケースがありましたら、お手数ですがご連絡をお願いいたします。</li>
<li>社員数が多い場合、PDFのダウンロードに時間がかかったり、ダウンロードできない場合がございます。現在改修中ですので少々お待ちください。</li>
<li>過去に端末を買い替えたり、複数の端末でログインした場合は、右上の「端末」から正しい端末を選択してください。</li>
</ul>
